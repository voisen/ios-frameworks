# ios-frameworks
编译后的各种frameworks, 欢迎各位贡献其他相关库文件

鉴于各种项目编译以下框架有一定难度，故在此提供以下编译好的库文件以供直接使用，目前编译完成的框架有：

- ffmpeg
- openssl
- lame
- ...

[ffmpeg-4.0.2]:https://raw.githubusercontent.com/voisen/ios-frameworks/master/frameworks/ffmpeg/4.0.2/ffmpeg-ios-4.0.2.zip
